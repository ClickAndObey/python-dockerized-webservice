# Python Dockerized Webservice

Example python webservice that is dockerized, and easily loads up in to PyCharm. It uses [Flask RESTPlus] and [Gunicorn]
for the webserver, and [Pytest] and [Pipenv] for testing.

## Quick Start

1. Run `make`
